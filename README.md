## 🎯 Что это и зачем?

**CommitCraft** — десктопное приложение для Git, которое помогает писать качественные коммиты в формате **Conventional Commits** с помощью нейросетей.

### 🔥 Зачем тебе это?

| Было | Стало |
|------|-------|
| `фикс` | `fix(auth): resolve login validation error` |
| `обнова` | `feat(api): add rate limiting for /users endpoint` |
| `ыва` | `docs(readme): update installation instructions` |
| `123` | `refactor(parser): extract error handling to separate module` |

Чистая история коммитов — это документация твоего проекта. CommitCraft делает её чистой без лишних усилий.

## ✨ Возможности

- 🤖 **Ручной AI-воркфлоу** — копируй промпт, вставляй в любой AI-чат, получай ответ.
- 🔘 **Три AI на выбор** — DeepSeek, Gemini, ChatGPT (один клик → открывается сайт).
- 📂 **Выбор репозитория** — работай с любым Git-проектом.
- 🔄 **Живой diff** — просмотр изменений до и после `git add`.
- ✏️ **Ручное редактирование** — правь сообщение перед коммитом.
- 📊 **Лимит 10 коммитов в день** — бесплатно, без регистрации, без API-ключей.
- ☕ **Поддержать автора** — кнопка прямо в приложении.

1. Перейди в [Releases](https://github.com/ego10sei/CommitCraft/releases)
2. Скачай `CommitCraft.exe`
3. Запусти и пользуйся
