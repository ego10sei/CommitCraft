## 🎯 Что это и зачем?

**CommitCraft** — это десктопное приложение, которое анализирует твои изменения в Git (`git diff`) и автоматически генерирует **осмысленные, профессиональные коммиты** в формате **Conventional Commits** через Google Gemini AI.

### 🔥 Зачем тебе это?

| Было | Стало |
|------|-------|
| `фикс` | `fix(auth): resolve login validation error` |
| `обнова` | `feat(api): add rate limiting for /users endpoint` |
| `ыва` | `docs(readme): update installation instructions` |
| `123` | `refactor(parser): extract error handling to separate module` |

**История коммитов — это документация твоего проекта.** Сделай её чистой без лишних усилий.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🤖 **AI‑генерация** | Gemini AI анализирует изменения и пишет коммит в Conventional Commits |

## 🚀 Быстрый старт

### Самый простой способ (для всех)

1. Перейди в [Releases](https://github.com/ego10sei/commitcraft/releases)
2. Скачай `CommitCraft.exe`
3. Запусти и пользуйся

### Для разработчиков (свой API-ключ)

1. Установи Python 3.10+
2. `pip install customtkinter`
3. Скачай исходники, вставь ключ
4. `python main.py`

🖱️ Как пользоваться?
-Выбери папку с Git‑репозиторием (кнопка «📁 Выбрать проект»)

-Добавь изменения в индекс (git add . – в терминале)

-Нажми «🔄 Обновить» – увидишь, что изменилось

-Нажми «✨ Сгенерировать сообщение» – ИИ напишет коммит

-Поправь, если нужно (можно редактировать)

-Нажми «💚 Сделать Commit» – изменения сохранятся в Git

🛠️ Технологии
Python 3.10+ – сердце проекта

CustomTkinter – современный UI

Gemini 1.5 Flash – генерация текста

GitPython – взаимодействие с Git (через subprocess)

PyInstaller – упаковка в .exe
