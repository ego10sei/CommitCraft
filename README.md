# ✨ CommitCraft — AI‑коммиты для разработчиков

[![Python](https://img.shields.io/badge/python-3.10%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![customtkinter](https://img.shields.io/badge/customtkinter-5.2.2-orange)](https://github.com/TomSchimansky/CustomTkinter)

**CommitCraft** — это десктопное приложение с ИИ, которое анализирует ваши изменения в Git и автоматически генерирует осмысленные сообщения для коммитов в стиле Conventional Commits.

🎯 **Цель:** сэкономить время и улучшить качество коммитов в ваших проектах.

✨ **Фишки:**
- 🤖 Генерация коммитов через Google Gemini AI
- 🎨 Красивый тёмный интерфейс в стиле GitHub
- 📊 Лимит 10 коммитов в день (бесплатно)
- ☕ Кнопка поддержки автора
- 🌓 Переключение тем (тёмная/светлая) – в планах
- 📁 Работа с любым Git-репозиторием
